# Road Accident Data Analysis

## Project Overview

This project analyzes road accident data to identify trends and factors contributing to road accidents. The goal is to uncover insights that could potentially improve road safety by focusing on high-risk areas, times, weather conditions, and other influential factors.

## Dataset

The dataset for this project, `Road Accident Data.csv`, includes various details on road accidents, such as location, date, time, weather conditions, and possible causes.

### Columns
- **Location**: The location of the accident.
- **Date**: Date when the accident occurred.
- **Time**: Time of the accident.
- **Weather Conditions**: Weather conditions at the time of the accident.
- **Cause**: The probable cause of the accident.
